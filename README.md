# 📄 문서 한글 자동 번역기

문서·웹페이지의 언어를 **자동 감지**하여 **한국어로 번역**하고, 원하는 형식으로 저장하는 로컬 도구입니다.
번역 엔진은 Google 번역(무료 엔드포인트)을 사용하며 인터넷 연결이 필요합니다.

## ✨ 기능

- **입력 형식**: `doc` · `docx` · `pdf` · `txt` · `xls` · `xlsx` · 웹주소(URL)
- **저장 형식**(문서류): `DOCX` · `PDF` · `JPEG` · `EXCEL` 중 여러 개 동시 생성
  - 엑셀 출력은 표를 추출해 숫자는 숫자형으로 저장하고 합계행에 `=SUM` 함수 자동 삽입
- **엑셀 입력**(xls/xlsx): 문자열 셀만 번역하고 **수식·숫자·서식은 그대로 보존**한 `_KO.xlsx` 저장
- **언어 자동 감지**: 영어·일어·중국어 등 무엇이든 한국어로
- 고유명사·약어(예: AspenTech, PSV)는 번역 불가 시 원문 유지
- docx의 Wingdings 등 **장식용 아이콘 글꼴은 건드리지 않아** 번역문이 깨지지 않음
- 필요한 외부 패키지는 실행 시 **자동 설치**

## 🚀 사용법

두 가지 방법 중 하나를 선택하세요.

### 1) `번역기.hta` (서버 불필요, 가장 간단)
더블클릭 → 파일 경로/URL 입력 → 저장 형식 체크 → **번역 시작**

### 2) `번역기 실행.bat` (웹 UI)
더블클릭하면 로컬 서버가 시작되고 브라우저가 열립니다. 브라우저에서 경로/URL 입력 후 사용.

> 결과 저장 위치: 로컬 파일은 **원본과 같은 폴더**, 웹주소는 **프로그램 폴더**에 `원본이름_KO.확장자` 로 저장됩니다.

## 🧩 요구 사항

- Windows + Python 3
- 자동 설치 패키지: `lxml`, `python-docx`, `beautifulsoup4`, `comtypes`, `PyMuPDF`, `openpyxl`, `xlrd`
  - 수동 설치: `pip install lxml python-docx beautifulsoup4 comtypes PyMuPDF openpyxl xlrd`
- `PDF`/`JPEG` 출력 및 `.doc` 입력에는 **MS Word**가 필요합니다.

## ⚠️ 참고 / 한계

- 사내 **DRM(예: Fasoo)** 으로 암호화된 PDF는 외부 도구가 읽지 못해 번역할 수 없습니다.
- Word로 만든 PDF가 DRM으로 보호되는 환경에서는 JPEG를 **번역 텍스트 이미지**로 대체 생성합니다.
- 스캔 이미지(텍스트 없는) PDF는 OCR이 없어 번역되지 않습니다.
- doc/pdf → docx 변환 시 복잡한 레이아웃·이미지는 단순화되며 번역 가독성을 우선합니다.
